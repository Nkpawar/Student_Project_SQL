# Student Course Enrollment System

## 📌 Project Description

This project manages students, courses, enrollments, exams, and results using MySQL.

## 🗂 Tables

* Students
* Courses
* Enrollment
* Exams
* Results

## 🔗 Relationships

* Students ↔ Courses → Many-to-Many (via Enrollment)
* Students → Exams → One-to-Many
* Exams → Results → One-to-One

## ⚙️ Features

* Student enrollment tracking
* Exam and result management
* SQL queries for analysis

## 🛠 Tech Used

* MySQL Workbench
* SQL

## 📊 Sample Queries

* JOIN operations
* GROUP BY
* CASE statements
